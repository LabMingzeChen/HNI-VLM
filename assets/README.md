# Assets

This folder holds graphics referenced from `README.md`.

## Required files

| filename | purpose | suggested source |
|---|---|---|
| `logo.png`            | Project logo (top of README) | Generate one, or use 🌿 + "HNI" text |
| `framework.png`       | Visual of Setting/Subject/Partner | Adapt Figure 4.1 of dissertation |
| `benchmark_chart.png` | F1 score by task by model | Adapt Figure 4.3 of dissertation |
| `examples_grid.png`   | 3 example images, one per category | Crop 3 Flickr images |

## How to add them

```bash
# Just drop the PNGs in this folder with the exact names above
cp ~/my_logo.png assets/logo.png
```

The README already references these names — no code changes needed once the files are here.

## Logo tips (free options)

- **Tldraw / Excalidraw** — sketch a leaf-and-lens combo.
- **Figma**             — clean typography only is fine.
- **PowerPoint**        — export a single slide at 1024×1024 PNG with transparent background.

Recommended size: **512×512 px**, transparent background.
